# 📡 IoT Telemetry API

A simple REST API built with **FastAPI** for receiving and managing IoT telemetry data.

This project demonstrates:

- Building a REST API with FastAPI
- Validating JSON input using Pydantic
- Storing telemetry data in memory
- Exposing multiple endpoints
- Using automatic interactive API documentation (Swagger)

---

## 🚀 Features

- `GET /` – API status
- `GET /health` – Health check
- `POST /telemetry` – Submit telemetry data
- `GET /telemetry` – Retrieve stored telemetry
- JSON validation
- In-memory storage
- Automatic Swagger documentation

---

## 🛠 Tech Stack

- Python 3.10+
- FastAPI
- Uvicorn
- Pydantic
